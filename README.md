# 🏘️ Community Needs Assessment Assistant

An AI-powered tool that helps nonprofit organizations analyze community survey responses and transform feedback into actionable insights.

## Overview

Community organizations frequently collect survey data but may not have the time or resources to manually analyze responses. This tool uses Claude AI to assist with identifying themes, community needs, and potential recommendations from survey data.

## Features

- Upload community survey CSV files
- Automatically identify survey questions
- Select multiple questions for analysis
- Generate AI-assisted community needs assessments
- Export reports as PDF documents

## Technology

- Python
- Streamlit
- Anthropic Claude API
- Pandas
- ReportLab

## How It Works

1. Upload survey response data
2. Define the purpose of the survey
3. Select questions to analyze
4. Claude analyzes response patterns
5. Generate a downloadable community assessment report

## Design Approach

The tool uses a human-in-the-loop approach. Users define the research purpose and select relevant survey data while Claude assists with analysis and synthesis.
